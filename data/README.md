# Training Data

Place `.txt` files of public-domain romantic novels in this directory.

## Recommended Sources

Download plain-text versions from [Project Gutenberg](https://www.gutenberg.org/):

| Title | Author | Gutenberg URL |
|-------|--------|---------------|
| Pride and Prejudice | Jane Austen | https://www.gutenberg.org/cache/epub/1342/pg1342.txt |
| Sense and Sensibility | Jane Austen | https://www.gutenberg.org/cache/epub/161/pg161.txt |
| Jane Eyre | Charlotte Brontë | https://www.gutenberg.org/cache/epub/1260/pg1260.txt |
| Wuthering Heights | Emily Brontë | https://www.gutenberg.org/cache/epub/768/pg768.txt |
| Persuasion | Jane Austen | https://www.gutenberg.org/cache/epub/105/pg105.txt |

## Download Example

```bash
cd data
curl -o pride_and_prejudice.txt https://www.gutenberg.org/cache/epub/1342/pg1342.txt
curl -o sense_and_sensibility.txt https://www.gutenberg.org/cache/epub/161/pg161.txt
curl -o jane_eyre.txt https://www.gutenberg.org/cache/epub/1260/pg1260.txt
```

All texts listed above are in the public domain in the United States.
